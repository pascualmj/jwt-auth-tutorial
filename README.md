# jwt-auth-tutorial
Simple NodeJS app using JWT Authentication. It is a simple API built using Express JS and MongoDB for data persistence. 

## Node Scripts
- start
- dev

## Dependencies
- express
- mongoose
- mongoose-unique-validator
- jsonwebtoken
- bcrypt
- dotenv
- morgan
- nodemon (dev dependency)
